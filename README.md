# Docker-venom
Docker version of ovh/venom - run your integration tests with efficiency in docker 

# Usage
```bash
docker run -v $(PWD):/app zhouzhuojie/docker-venom venom help
docker run -v $(PWD):/app zhouzhuojie/docker-venom venom version
```
